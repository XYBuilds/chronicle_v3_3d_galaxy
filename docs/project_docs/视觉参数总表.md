## 1. 渲染器与相机

| 参数 / 行为 | 当前值 | 位置 | 说明 |
|-------------|--------|------|------|
| 场景背景色 | `0x000000` | `scene.ts`（`scene.background`） | 纯黑 |
| 透视 FOV / near / far | `50` / `0.05` / `1e6` | `scene.ts` `PerspectiveCamera` | |
| 相机固定朝向 | `GALAXY_CAMERA_EULER` = `(0, π, 0)` YXZ | `camera.ts` `GALAXY_CAMERA_EULER` | 轴平行沿 +Z 观察 |
| 宏观相机 standoff | `zCamDistance = 30`（写入 store） | `scene.ts` 常量 + `galaxyInteractionStore` | idle：`camera.z = zCurrent - zCamDistance` |
| `zVisWindow` 初始 | `1` | `scene.ts` + store | shader Z 可视 slab 宽度 |
| `zCurrent` 初始 | `z_range` 较小端 `zLo` | `scene.ts` | 首屏对齐「时间原点」 |
| DPR 上限 | `min(devicePixelRatio, 2)` | `scene.ts` `setPixelRatio` / `resize` | 与 `uPixelRatio` 同步 |
| 输出色彩空间 | `SRGBColorSpace` | `scene.ts` | |
| WebGL2 | 必需；否则抛错 | `scene.ts` `renderer.capabilities.isWebGL2` | `gl_InstanceID` / 双 mesh |
| 选中飞入 / 退出 | `700 ms` / `450 ms` | `scene.ts` `SELECT_MS` / `DESELECT_MS` | `easeOutCubic` |
| XY 钳制 padding | `0.08` | `scene.ts` 调用 `clampGalaxyCameraXY`；`camera.ts` 默认 | |
| Truck / pedestal | `0.02` world/px | `camera.ts` `truckPedestalSpeed` 默认 | |
| 滚轮 Z | `0.15` × 幅度缩放 | `camera.ts` `zScrollSpeed` 默认 | macro 时改 store `zCurrent` |
| Perlin 焦点相机 | `movie.z - 1` | `camera.ts` `FOCUS_PERLIN_CAMERA_STANDOFF` | `setFocusCameraPosition` |

## 2. 宏观星系 · 双 InstancedMesh（P8.4）

| 参数 | 当前值 | 位置 | 说明 |
|------|--------|------|------|
| 几何 | Idle：`Icosahedron(1,0)`；Active：`Icosahedron(1,1)` | `galaxyMeshes.ts` | 共享 instance 属性 |
| `uSizeScale` 初值 | `0.3`（`DEFAULT_GALAXY_U_SIZE_SCALE`） | `galaxyMeshes.ts` | 注释对齐旧 Points 标定 |
| `uActiveSizeMul` / `uBgSizeMul` | `0.02` / `0.002` | `galaxyMeshes.ts` `makeSharedUniforms` | 生产挂载；Storybook lab args 曾用 `uBgSizeMul: 0.001`，以本处为准 |
| `uLMin` / `uLMax` / `uChroma` | `0.4` / `0.85` / `0.15` | `galaxyMeshes.ts` | OKLab 映射 |
| `uFocusedInstanceId` | 初值 `-1` | `scene.ts` 挂载后 | |
| Idle 材质 | `transparent`、无 `depthWrite` | `galaxyMeshes.ts` | `depthTest: true` |
| Active 材质 | 不透明、`alphaTest: 0.01` | `galaxyMeshes.ts` | `depthWrite: true` |
| `renderOrder` | idle `0`，active `1`；星球在 scene 设为 `2` | `galaxyMeshes.ts` / `scene.ts` | |
| Z slab 过渡宽 | `W = uZVisWindow * 0.2` | `galaxyIdle.vert.glsl`、`galaxyActive.vert.glsl` | `inFocus` |
| Idle alpha | `clamp(1.0 - vInFocus, 0.14, 0.95)` | `galaxyIdle.frag.glsl` | |
| Active 片元 | `vec4(vColor, 1.0)` | `galaxyActive.frag.glsl` | |

**共享 uniform**：`uPixelRatio`, `uZCurrent`, `uZVisWindow`, `uSizeScale`, `uActiveSizeMul`, `uBgSizeMul`, `uLMin`, `uLMax`, `uChroma`, `uFocusedInstanceId`。

## 3. 拾取与屏幕半径

| 参数 | 当前值 | 位置 | 说明 |
|------|--------|------|------|
| 点击抖动容差 | `6` px | `interaction.ts` `CLICK_MAX_MOVE_PX` | |
| 点击 slab 门限 | `inFocus > 0.5` | `screenRadius.ts` `pickClosestActiveMovieAlongRay` | 与 shader 同构 |
| 选中球半径回退 | `clamp(worldSpan×0.014, 0.07, worldSpan×0.05)` | `screenRadius.ts` `resolveSelectionWorldRadius` | `rActive` 极小时 |
| Active 世界半径 | `inFocus × uSizeScale × uActiveSizeMul × size` | `screenRadius.ts` `computeActiveWorldRadius` | 与 `galaxyActive.vert` 一致 |

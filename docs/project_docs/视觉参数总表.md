## 2. 宏观星系 · 双 InstancedMesh（P8.4 + Phase 10）

| 参数 | 当前值 | 位置 | 说明 |
|------|--------|------|------|
| 几何 | Idle：`Icosahedron(1,0)`；Active：`Icosahedron(1,1)` | `galaxyMeshes.ts` | 共享 instance 属性 |
| `uSizeScale` 初值 | `0.3`（`DEFAULT_GALAXY_U_SIZE_SCALE`） | `galaxyMeshes.ts` | 注释对齐旧 Points 标定 |
| `uActiveSizeMul` / `uBgSizeMul` | `0.02` / `0.002` | `galaxyMeshes.ts` `makeSharedUniforms` | 生产挂载；Storybook lab args 曾用 `uBgSizeMul: 0.001`，以本处为准 |
| `uLMin` / `uLMax` / `uChroma` | `0.2` / `1.0` / `0.15` | `galaxyMeshes.ts` | OKLab **L** 端点（**P10.1**） |
| **P10.1** `uHighRatingT` / `uHighTierTRangeScale` / `uLightnessRatingExponent` | `0.85` / `0.4` / `3.0` | `galaxyMeshes.ts`；顶点着色器内作用在 `voteNorm`→压缩 `t`→`pow`→`L` | 高分段斜率压缩 + 非线性；见 `galaxyIdle.vert.glsl` / `galaxyActive.vert.glsl` |
| **P10.2** `uDistanceFalloffK` | `0.0001` | `galaxyMeshes.ts` | 世界坐标 **`dz`** 定义见下行 |
| **P10.2** `uDistanceFalloffMode` | `1`（on） | `galaxyMeshes.ts` | `0` = 关闭衰减与 idle 高 alpha 分支；`1` = 启用 |
| **P10.2** `vDistFalloff` | \(1/(1 + k \cdot dz^2)\) | `galaxyIdle.vert.glsl`、`galaxyActive.vert.glsl` | `zHi = uZCurrent + uZVisWindow`，`dz = max(0, aZ - zHi)`（**仅超出可视窗上沿的未来 Z** 变暗；条带内 `dz=0` → 不衰减） |
| `uFocusedInstanceId` | 初值 `-1` | `scene.ts` 挂载后 | |
| Idle 材质 | `transparent`、无 `depthWrite` | `galaxyMeshes.ts` | `depthTest: true` |
| Active 材质 | 不透明、`alphaTest: 0.01` | `galaxyMeshes.ts` | `depthWrite: true` |
| `renderOrder` | idle `0`，active `1`；星球在 scene 设为 `2` | `galaxyMeshes.ts` / `scene.ts` | |
| Z slab 过渡宽 | `W = uZVisWindow * 0.2` | `galaxyIdle.vert.glsl`、`galaxyActive.vert.glsl` | `inFocus` |
| Idle alpha（`galaxyIdle.frag.glsl`） | 见下 | — | **`m = float(uDistanceFalloffMode)`**：`m=0` 时 `alpha = clamp(1.0 - vInFocus, 0.14, 0.95)`（与 P8.4 一致）；`m=1` 时 `alpha = 0.95`（固定高 alpha，避免与 Z 衰减双重压暗/透明） |
| Active 片元 | `vColor * mix(1, vDistFalloff, m)`，不透明 | `galaxyActive.frag.glsl` | `m` 同上；`m=0` 时无 Z 衰减 |

**共享 uniform**（与 `galaxyMeshes.ts` `makeSharedUniforms` 一致）：`uPixelRatio`, `uZCurrent`, `uZVisWindow`, `uSizeScale`, `uActiveSizeMul`, `uBgSizeMul`, `uLMin`, `uLMax`, `uHighRatingT`, `uHighTierTRangeScale`, `uLightnessRatingExponent`, `uDistanceFalloffK`, `uDistanceFalloffMode`, `uChroma`, `uFocusedInstanceId`。

## 附录：方法说明

- **多值字段拆分**: `genres`、`production_countries`、`spoken_languages` 为受控词表，直接按 `,`  拆分；`production_companies`、`cast`、`director`、`writers`、`producers`、`music_composer` 采用 `smart_split` 后缀回合策略，将 `Inc.`/`Ltd.`/`Jr.`/`Sr.` 等误切片段拼回前一实体（影响约 500 条记录）
- **零值 vs 缺失**: 数值字段的空值数和零值数分开统计。`budget`/`revenue` 中零值实为"TMDB 未收录"，不应与 NaN 混淆
- **偏度 / 峰度**: 使用 pandas 默认的 Fisher 定义（正态分布峰度 = 0）
- **滑动窗口（365/30 天）**: 对每个窗口起点日 `d`，计数满足 `d ≤ release_date < d + W` 的影片（左闭右开，连续 `W` 个日历日）；在数据集首日与末日之间逐日滑动，用按日上映数量的前缀和计算。与自然年、自然月的分桶统计口径不同
- **可复现**: `python scripts/tmdb_data_report.py public/data/raw/tmdb2025.csv "project docs/personal/TMDB 2025 Dataset Report.md"`

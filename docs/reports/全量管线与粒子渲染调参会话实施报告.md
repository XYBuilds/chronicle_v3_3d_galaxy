## 5. Three.js 场景与粒子（当前实现）

### 5.1 Bloom（可恢复）

- **文件**：`frontend/src/three/scene.ts`  
- **`UnrealBloomPass`**：构造时 **strength = `0.0`**（关闭泛光，便于观察实体）；`radius`、`threshold` 仍保留，便于恢复调参。  
- **调试**：`window.__bloom.strength` / `.radius` / `.threshold` 与 **`log()`**；场景卸载时删除 `window.__bloom`。

### 5.2 片元：`point.frag.glsl`

- 圆形：`r = length(gl_PointCoord - 0.5) * 2.0`，**`r > 1.0` → `discard`**。  
- **硬边**：`inner = 0.88`，**`mix(vec3(1.0), vColor, step(r, inner))`** — 内盘为流派色，外环为白色描边；**`alpha = 1.0`**。  
- **`vEmissive` varying**：保留以满足与顶点 varying 的链接约定；**当前不参与 RGB**（原「关闭发光 / 可读性」路径的延续）。

### 5.3 顶点：`point.vert.glsl`

- **`gl_PointSize = size * uPixelRatio * (500.0 / dist) * uSizeScale`**，其中 **`dist = max(0.001, -mvPosition.z)`**，体现**远小近大**的透视缩放。  
- **无 shader 内 clamp**；硬件仍可能对 `gl_PointSize` 做实现相关限制。

### 5.4 材质与深度：`frontend/src/three/galaxy.ts`

| 属性 | 值 | 说明 |
|------|-----|------|
| `uniforms.uSizeScale` | 默认见 `DEFAULT_POINT_SIZE_SCALE`（**0.3**） | 全局点大小倍率。 |
| `transparent` | **`false`** | 与不透明白边圆片一致。 |
| `depthTest` | **`true`** | 深度测试开启。 |
| `depthWrite` | **`true`** | 写入深度，修正「远片盖住近片」的典型错误。 |

### 5.5 控制台：`window.__galaxyPointScale`

- **文件**：`frontend/src/three/scene.ts`  
- **`scale`**：读写 **`material.uniforms.uSizeScale`**。  
- **`log()`**：打印当前倍率。  
- **`dispose`**：若当前实例仍挂在 `window.__galaxyPointScale` 上则删除，避免泄漏。

**示例**：

```js
window.__galaxyPointScale.scale = 0.2
window.__galaxyPointScale.log()
```

### 5.6 `createGalaxyPoints` 签名

```ts
createGalaxyPoints(movies, pixelRatio, sizeScale?)
```

第三参数可选；省略时使用 **`DEFAULT_POINT_SIZE_SCALE`**。

## 6. Bloom / 过曝相关参数（梳理结论，便于后续调回「星云辉光」）

以下在会话中已系统梳理；**当前代码刻意未锁定生产向 Bloom**。未来定稿生产辉光时，恢复星云风格需联动考虑下列旋钮（与计划「Bloom 冻结、标定推迟」一致）。

| 层级 | 位置 | 参数或常量 |
|------|------|------------|
| 后处理 | `scene.ts` | `UnrealBloomPass`：**strength, radius, threshold**；分辨率向量。 |
| 片元（历史/可恢复） | `point.frag.glsl` | 光晕 `glow`、`vEmissive` 乘子、`smoothstep` 与 **`0.4 + 0.9*core + 1.4*glow`** 等 HDR 标定。 |
| 顶点 | `point.vert.glsl` | **`500.0`** 透视尺度系数；**`uSizeScale`**。 |
| 数据导出 | `export_galaxy_json.py` | **`--emissive-min` / `--emissive-max`**；**`--size-min` / `--size-max`**。 |
| 渲染器默认 | `scene.ts`（未显式设置） | `toneMapping`、`toneMappingExposure`、`outputColorSpace` 等 Three.js 默认会影响最终观感。 |
